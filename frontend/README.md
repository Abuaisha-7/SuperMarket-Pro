# SuperMarket Inventory Management System

A modern React-based frontend for managing supermarket inventory, sales, and operations. Built with Vite, React, and Tailwind CSS.

## Features

- **Home Page**: Welcome dashboard with feature overview
- **Login Page**: User authentication interface
- **Inventory Management**: Product tracking with stock levels, categories, and status
- **Sales Dashboard**: Sales tracking, revenue analytics, and top products
- **Responsive Design**: Mobile-friendly interface
- **Modern UI**: Clean, professional design with Tailwind CSS

## Tech Stack

- **React 18**: Modern React with hooks
- **Vite**: Fast build tool and development server
- **React Router DOM**: Client-side routing
- **Tailwind CSS**: Utility-first CSS framework
- **PostCSS**: CSS processing

## Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd supermarket-app/frontend
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

### Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
